# 📌 Overview

This repository showcases an **end-to-end Document Control Management System (DCMS)** developed as a practical portfolio project based on real-world construction document control practices.

The project simulates the complete document lifecycle followed in engineering and construction projects, covering document creation, numbering, review, approval, revision control, transmittals, filing, EDMS administration, audit readiness, and KPI monitoring while aligning with **ISO 9001:2015 Documented Information** requirements.

---

# 🏗️ Business Problem

Large engineering and construction projects generate thousands of documents throughout their lifecycle.

Without an effective document control system, organizations commonly face:

- Duplicate documents
- Incorrect revisions
- Uncontrolled document distribution
- Poor document traceability
- Lost project records
- Approval delays
- Audit nonconformities
- Compliance risks

A structured Document Control Management System ensures document integrity, traceability, compliance, and efficient collaboration across all project stakeholders.

# 🚀 Project Workflow

### Phase 1 — Controlled Documents
Established document control procedures, identified controlled documents, and created document classification standards.

↓

### Phase 2 — Document Numbering
Designed a standardized numbering convention using project, discipline, sequence, and revision codes.

↓

### Phase 3 — Document Registers
Built master registers to manage document status, revisions, ownership, and traceability.

↓

### Phase 4 — Review & Approval
Developed structured review workflows, approval processes, document change requests, and revision control.

↓

### Phase 5 — Transmittals & Distribution
Implemented controlled document distribution through transmittals, distribution matrices, acknowledgements, and recall processes.

↓

### Phase 6 — Filing, Storage & Retrieval
Designed electronic and physical filing systems, retrieval logs, retention schedules, and disposal registers.

↓

### Phase 7 — EDMS Administration
Created EDMS governance including user access, permission management, audit trails, backups, recovery procedures, and workflow management.

↓

### Phase 8 — Audit Readiness & KPI Monitoring
Prepared audit documentation, self-assessment forms, KPI dashboards, NCR tracking, CAPA management, and continual improvement workflows.
